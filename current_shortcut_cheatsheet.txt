## GENERAL:

 - LEADER p: paste over text without losing the initial paste from the buffer
 - LEADER #: Switch between relative and absolute line numbers
 - LEADER ': Switch between paste and nopaste
 - LEADER z: fuzzy (FZF) search


## Window management:

 - CTRL SHIFT k: toggle maximized
 - CTRL SUPER k: maximize vertically
 - CTRL SUPER l: maximize horizontally
 - CTRL SUPER j: minimize
 - CTRL SUPER c: move to center

 - CTRL s: begin resize
 - CTRL SHIFT m: begin move

 - CTRL y: move to corner nw
 - CTRL o: move to corner ne
 - CTRL b: move to corner sw
 - CTRL m: move to corner se

 - CTRL h: move to side w
 - CTRL k: move to side n
 - CTRL j: move to side s
 - CTRL l: move to side e

 - CTRL SHIFT SUPER h: move to monitor left
 - CTRL SHIFT SUPER l: move to monitor right


## Vimspector:

 - SUPER d b: Launch Vimspector
 - SUPER d e: Reset
 - SUPER d c: Continue

 - SUPER d t: Toggle breakpoint
 - SUPER d T: Clear breakpoints

 - SUPER d o: Step out
 - SUPER d i: Step in
 - SUPER d O: Step over
 - SUPER d r: Restart

## COC:

### GENERAL

 - CTRL + SPACE: trigger completion

 - K : Show documentation in preview window

 - LEADER r n: Rename symbol

 - LEADER f: Format selected code

### DIAGNOSTICS

 - [ g : Go to previous diagnostic

 - ] g : Go to next diagnostic

### Code navigation

 - g d : Go to definition

 - g y : Go to type definition

 - g i : Go to implementation

 - g r : Go to references

### Code Actions

 - LEADER a: Apply codeAction to the selected region

 - LEADER a c: Apply codeAction to current buffer

 - LEADER q f: AutoFix to the problem on the current line

### Navigate windows/popups

 - CTRL + f: Scoll forward 
 - CTRL + b: Scroll backward

### CoCList

 - SPACE + a: Show diagnostics
 - SPACE + e: Manage extensions
 - SPACE + c: Show commands
 - SPACE + o: Find symbol of current document
 - SPACE + s: Find workspace symbols
 - SPACE + j: Do default action for next item
 - SPACE + k: Do default action for previous item
 - SPACE + p: Resume latest CoCList
